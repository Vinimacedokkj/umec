# 🌍 Site U.M.E.C. - União Mundial de Ministros do Bem e Empreendedores dos Continentes

## 📋 Descrição

Site institucional completo para a U.M.E.C., uma organização não governamental que congrega ministros, líderes e empreendedores engajados na promoção do bem-estar social, solidariedade e desenvolvimento econômico em diversos continentes.

## ✨ Características

- **Design Responsivo**: Funciona perfeitamente em todos os dispositivos
- **Cores Personalizadas**: Seguindo exatamente a paleta de cores especificada
- **Fontes Elegantes**: Playfair Display, Lato, Raleway e Inter do Google Fonts
- **Animações Suaves**: Efeitos visuais modernos e profissionais
- **Navegação Intuitiva**: Menu fixo com scroll suave
- **Funcionalidades Interativas**: Botões de doação, copiar PIX, etc.

## 🎨 Paleta de Cores

```css
:root {
  --cor-principal: #A28B42;      /* Dourado principal */
  --fundo-pagina: #F3F1EB;       /* Fundo bege claro */
  --cor-secundaria: #6D5E2E;     /* Dourado escuro */
  --cor-titulo: #2E2C26;         /* Preto para títulos */
  --cor-subtitulo: #4B4739;      /* Marrom escuro */
  --cor-texto: #5A564C;          /* Marrom para texto */
  --cor-clara: #FFFFFF;          /* Branco */
}
```

## 🚀 Como Usar

### 1. **Instalação**
```bash
# Clone ou baixe os arquivos
# Abra o index.html em um navegador
```

### 2. **Personalização**

#### **Cores**
Edite as variáveis CSS no arquivo `style.css`:
```css
:root {
  --cor-principal: #A28B42;  /* Sua cor principal */
  --fundo-pagina: #F3F1EB;   /* Seu fundo */
  /* ... outras cores */
}
```

#### **Conteúdo**
- **HTML**: Edite o `index.html` para alterar textos e informações
- **Imagens**: Substitua as imagens na pasta `img/`
- **Links**: Atualize URLs das redes sociais e contatos

#### **Funcionalidades**
- **JavaScript**: Modifique o `script.js` para novas funcionalidades
- **Formulários**: Adicione campos de contato se necessário

### 3. **Deploy**
- Faça upload dos arquivos para seu servidor web
- Certifique-se de que todos os arquivos estão na mesma pasta
- Teste em diferentes dispositivos e navegadores
